# Проект «Лайк за мысль» 💡

Платформа для сбора и ранжирования идей с голосованием и модерацией. Хорошие идеи не теряются — они получают лайки и реализацию.

## Технологический стек

### Основной стек (вариант 36)
- **Frontend**: React + TypeScript + Vite + Redux Toolkit
- **Backend**: NestJS + TypeScript + Prisma + PostgreSQL
- **Аутентификация**: JWT + bcrypt
- **Документация API**: Swagger/OpenAPI
- **Контейнеризация**: Docker + Docker Compose
- **CI/CD**: GitHub Actions

## Быстрый старт

### Требования
- Docker и Docker Compose
- Node.js 18+ (для локальной разработки)
- PostgreSQL 14+ (или используйте Docker)

## 📚 API Документация

После запуска backend приложения, автоматически генерируемая Swagger документация доступна по адресу:
**http://localhost:3001/api/docs**
